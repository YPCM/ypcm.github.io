### What is Romifleur?
**Romifleur** is a modern, user-friendly desktop application designed to simplify the process of finding and downloading retro game ROMs. Built with Python and CustomTkinter, it offers a sleek interface to browse extensive catalogs for various classic consoles, providing a streamlined alternative to navigating complex archive websites manually.

### Features
*   **Modern GUI**: A clean, dark-themed interface powered by CustomTkinter enhances your experience.
*   **Multi-Console Support**: Access libraries for multiple systems, including NES, SNES, N64, GameCube, PS1, PSP, Sega Genesis, and Dreamcast.
*   **Rich Metadata Integration**: View game thumbnails, descriptions, release dates, and more directly in the app, thanks to [**TheGamesDB**](https://thegamesdb.net/).
*   **Smart Search & Filtering**:
    *   Use a real-time search bar to find games quickly.
    *   Easily toggle between regions such as Europe, USA, and Japan.
